# CodeForces-800
This repository contains codeforces 800 level problems solutions with explaination. The explanation will help coders to build logic on their own and solve higher level problems more efficiently in the future. It is highly recommended not to copy-paste the codes from the repository just to get higher problem solved count in codeforces. The codes are written in <i>Python</i>. <br>

## Pre-requisite
<ul>
  <li> Basic coding knowledge in python.</li>
  <li> Knowledge about Jupyter Notebook.</li>
  <li> Knowledge about how .ipynb files work (not mandatory)</li>
</ul>
